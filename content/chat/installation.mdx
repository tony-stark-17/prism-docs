## Installation
- **Replace the ```cl_chat.lua``` & ```sv_chat.lua``` from the default CHAT resource with the ones on _INSTALLATION folder**
- **Add Permissions**: ```add_ace group.admin prismchat allow```
- **[OPTIONAL] Add the steam API on the `server.cfg`**
- **[OPTIONAL] Add the discord bot token API on `sv_utils.lua`**
- **Drag-and-drop our script** : Drag `prism_chat` in to your `resources` directory and add `ensure prism_chat` to your `server.cfg`
- **Restart your server**.
