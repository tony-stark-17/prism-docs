## Add the compatibility script

Add `@prism_uipack/init.lua` to the shared_scripts list inside your resourceâ€™s `fxmanifest.lua`.

```lua copy
shared_scripts {
    '@ox_lib/init.lua',
    '@prism_uipack/init.lua'
}
```

<Callout type="warning">
  Do not add this into the `ox_lib` resource. You have to put it in every resource that is using ox_lib itself.
</Callout>
